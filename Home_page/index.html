HTML (HyperText Markup Language)


Definition

The standard language used to create the structure and content of web pages. It defines elements like headings, paragraphs, images, and links to build the framework of a website.

Structure of Web Pages

HTML provides the basic structure of a webpage using various elements.

Tags and Elements

HTML uses a tag-based system like <h1>, <p>, and <div>.

Links and Navigation

HTML allows linking between pages using anchor tags (<a>).

Semantic Elements

HTML5 includes tags like <header>, <footer>, and <section>.

Media Integration

Embed multimedia content using <img>, <video>, and <audio>.

CSS (Cascading Style Sheets)


Definition

A stylesheet language used to control the presentation and layout of HTML elements. It defines the visual style, such as colors, fonts, spacing, and positioning, allowing for responsive design.

Presentation and Styling

CSS styles the appearance of HTML elements.

Selectors and Properties

CSS applies styles through selectors like class, id, or element name.

Responsive Design

Media queries in CSS help adapt the layout to different screen sizes.

Box Model

CSS organizes elements using the box model: content, padding, border, and margin.

Flexbox and Grid

Layout techniques like Flexbox and CSS Grid create complex, responsive designs.

JavaScript


Definition

A programming language that adds interactivity and dynamic behavior to websites. It enables features like form validation, animations, and interactive elements, running directly in the browser.

Interactivity

JavaScript adds interactive behavior to web pages.

Client-Side Scripting

JavaScript runs in the browser and manipulates the DOM dynamically.

Variables and Data Types

JavaScript uses variables to store data, including strings, numbers, and booleans.

Event Handling

JavaScript responds to user events like clicks and key presses.

APIs and Libraries

JavaScript can interact with web APIs and utilize third-party libraries.
